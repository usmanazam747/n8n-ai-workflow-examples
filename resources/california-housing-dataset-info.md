# California Housing Dataset

This dataset contains information about California housing, including features such as location, median house value, median income, and more. It is commonly used for real estate analytics and machine learning experiments.

- **Source:** [Kaggle - California housing](https://www.kaggle.com/datasets/sooyoungher/california-housing)
- **Format:** CSV
- **Typical columns:** longitude, latitude, housing_median_age, total_rooms, total_bedrooms, population, households, median_income, median_house_value

## Usage
- Use this dataset to simulate property listings for workflow automation.
- Import the CSV into your n8n workflow for data centralization, enrichment, or notification scenarios.

## Download
Visit the Kaggle link above to download the CSV file and place it in the `/resources` folder.
